# 金珂重点关注公司新闻动态

**独立网站**: https://lovinglaura.github.io/company-news/

## 🎯 项目简介

深度分析谷歌、英伟达、特斯拉、腾讯、茅台等公司的新闻，特别关注可能影响股价的重要事件。

## 🌐 网站特点

- **每日自动更新**: 每天北京时间早上9:00自动更新
- **三级搜索策略**: 实时新闻(24h) + 重要新闻(3d) + 深度分析(7d)
- **智能价值评分**: 1-10分筛选高质量新闻
- **股价影响评估**: 评估新闻对股价的短期和长期影响
- **关键数据提取**: 提取所有量化信息

## 📊 覆盖公司

| 公司 | 代码 | 关注领域 |
|------|------|----------|
| 谷歌 | GOOGL | AI、搜索、云计算、财报 |
| 英伟达 | NVDA | GPU、AI芯片、数据中心 |
| 特斯拉 | TSLA | 电动车、自动驾驶、能源 |
| 腾讯 | 0700.HK | 游戏、社交、投资 |
| 茅台 | 600519.SS | 白酒、消费、高端品牌 |

## 🚀 快速开始

### 本地运行

```bash
# 1. 安装依赖
npm install

# 2. 抓取和分析新闻
npm run fetch

# 3. 生成HTML页面
npm run generate

# 4. 启动本地服务器（可选）
npm start
```

### 自动更新

网站每天自动更新（北京时间早上9:00），无需人工干预。

## 🔧 技术架构

- **自动化抓取**: 使用coze-web-search API
- **静态托管**: GitHub Pages
- **前端框架**: Tailwind CSS
- **自动化部署**: GitHub Actions

## 📈 价值评分标准

- **8-10分 (🔥必读)**: 重大财报、盈利预警、产品发布、技术突破
- **6-7分 (⭐推荐)**: 重要合作、订单、市场扩张、监管动态
- **1-5分 (📰参考)**: 一般新闻、传闻、分析、行业动态

## 🎨 设计特点

- 响应式布局，支持手机、平板、电脑
- 公司分类颜色标识
- 股价影响级别标识
- 深度分析内容展示

## 🔄 自动化流程

1. **定时触发**: 每天UTC 1:00（北京时间9:00）
2. **新闻抓取**: 使用coze-web-search API获取最新新闻
3. **深度分析**: 分析逻辑链条和股价影响
4. **HTML生成**: 生成美观的展示页面
5. **自动部署**: 推送到GitHub Pages

## 🔐 环境配置

### GitHub Secrets配置

需要在GitHub仓库中配置以下Secrets：

1. **COZE_API_KEY**: coze-web-search API密钥
2. **GITHUB_TOKEN**: GitHub访问令牌（自动生成）

### 本地环境配置

创建`.env`文件：

```bash
COZE_API_KEY=your_api_key_here
```

## 🤖 自动化更新

### GitHub Actions工作流

每天自动运行以下步骤：

1. **Checkout代码**: 获取最新代码
2. **安装依赖**: 安装Node.js依赖
3. **抓取新闻**: 使用API获取最新公司新闻
4. **生成HTML**: 创建更新后的网站页面
5. **提交推送**: 自动提交并推送到仓库
6. **部署**: GitHub Pages自动部署

## 📊 监控指标

- **更新频率**: 每天一次

## 🐛 故障排除

### 常见问题

1. **API密钥无效**: 检查COZE_API_KEY是否正确配置
2. **新闻抓取失败**: 检查网络连接和API配额
3. **HTML生成错误**: 检查数据格式和脚本逻辑

### 解决方案

1. 重新生成API密钥
2. 检查GitHub Secrets配置
3. 查看GitHub Actions运行日志
